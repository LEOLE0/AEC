Je vais concevoir et développer le site web complet pour **AEC – AUDIT ÉNERGÉTIQUE CONSEIL** en suivant strictement vos directives : design ultra-épuré, corporate, moderne et parfaitement optimisé pour le mobile.

Voici le plan d'action détaillé pour atteindre l'excellence attendue :

## 1. Architecture Technique & Stack (Best Practices 2026)
Nous partirons sur une stack robuste, performante et SEO-friendly.
*   **Framework** : Next.js 15 (App Router) - Pour le SSR, le SEO et la scalabilité.
*   **Langage** : TypeScript - Pour un code typé, maintenable et sans erreur.
*   **Styling** : Tailwind CSS - Pour le Design System sur-mesure et la rapidité d'exécution.
*   **UI Components** : Shadcn/ui (radix-ui) personnalisé - Pour des composants accessibles, cleans et robustes.
*   **Icônes** : Lucide React - Style filaire, professionnel et léger.
*   **Formulaires** : React Hook Form + Zod - Pour la validation stricte des données (Étude en ligne).
*   **Architecture** : Modulaire (`/components`, `/lib`, `/app`).

## 2. Design System & Identité Visuelle (Conforme Charte)
Je vais définir les fondations graphiques avant de coder les pages.
*   **Palette Couleurs** :
    *   *Primary Blue* : Un bleu nuit profond, institutionnel et rassurant (ex: Navy Blue).
    *   *Accent Green* : Un vert nature/énergie, vibrant mais sérieux (ex: Emerald/Forest).
    *   *Neutrals* : Blanc pur, gris clairs pour les fonds, gris foncés pour le texte (lisibilité maximale).
*   **Typographie** : Sans-serif moderne (ex: `Inter` ou `Plus Jakarta Sans`) pour une lisibilité parfaite sur tous écrans.
*   **Formes** :
    *   *Radius* : Arrondis prononcés pour les boutons (Pill shape) et modérés pour les cartes (Rounded-xl) pour l'aspect moderne.
    *   *Spacing* : Espaces généreux (Whitespace) pour l'aspect "Ultra épuré".
*   **Navbar** : Flottante, centrale, effet "glassmorphism" subtil, ultra-moderne.

## 3. Arborescence & Contenu des Pages
Je développerai les pages suivantes avec une structure HTML sémantique pour le SEO.

*   **Accueil (Home)**
    *   *Hero Section* : Image corporate haute qualité, Tagline forte, CTA principal ("Demander un devis").
    *   *Confiance* : Logos partenaires/institutionnels.
    *   *Aperçu Expertises* : 3 piliers clés.
    *   *Chiffres clés* : Impact énergétique.

*   **Expertises**
    *   Grille détaillée des services (Audit réglementaire, DPE, Conseil technique).
    *   Approche technique et normative.

*   **Méthodologie**
    *   Timeline verticale ou horizontale expliquant le processus (1. Audit -> 2. Analyse -> 3. Rapport -> 4. Accompagnement).

*   **Étude en ligne (Lead Gen)**
    *   Formulaire multi-étapes (Wizard) pour qualifier le besoin (Type de bâtiment, surface, localisation).
    *   UX fluide pour maximiser la conversion.

*   **À propos**
    *   Valeurs de l'entreprise (Rigueur, Innovation, Durabilité).
    *   L'équipe (Photos pro).

*   **Contact**
    *   Coordonnées complètes.
    *   Formulaire de contact rapide.

1.  **Setup** : Initialisation du projet et configuration du Design System (Tailwind).
2.  **Composants de base** : Création des briques (Boutons, Cards, Inputs, Navbar, Footer).
3.  **Pages** : Intégration page par page avec contenu "Lorem Ipsum" orienté métier (textes réalistes).
4.  **Responsive** : Vérification et ajustement méticuleux pour mobile (Mobile First approach).
5.  **Optimisation** : SEO (Metatags), Performance (Images), Accessibilité.
